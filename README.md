## Install

```bash
## mkdocs
# see: https://www.mkdocs.org/#installation
pip install --upgrade pip
pip install mkdocs

## Theme: meterial
# https://squidfunk.github.io/mkdocs-material/
pip install mkdocs-material
```

## Local Debugging

```bash
mkdocs serve
```
